# Commons Greenhouse Calculator Secure

Community-first, offline-capable calculator and blueprint generator for the **Commons Greenhouse System (CGS-C1)** / **Sistema de Invernadero Comunal (SIC-C1)**.

## Principles

- Free for community use
- Open plans
- Low-tech first
- Anchoring and safety are non-negotiable
- Bilingual by default
- No accounts, no tracking, no external dependencies

## Included

- `index.html` - secure shell with strict CSP
- `app.js` - client-side logic only
- `styles.css` - local styles only
- `manifest.webmanifest` - installable PWA metadata
- `sw.js` - offline service worker

## Canonical Rules

1. Structure permanent, skin replaceable.
2. Earth + air + thermal mass before gadgets.
3. Design for local labor and local materials.
4. Open designs, community-safe implementation.
5. Anchoring, uplift resistance, and wind hardening come first.

## Community Use

You may use this calculator freely for:

- community gardens
- mutual aid projects
- school and neighborhood builds
- personal and family food production

## Notes

This is the secure branch of the calculator concept. It is intended as a public-facing community tool and a canonical reference point for future CGS/SIC cells, including aquaponics, solar, and multi-cell expansion.
